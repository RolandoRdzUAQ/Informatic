# Respuestas a Actividad 1: Comprensión conceptual

**1.- ¿Cuál es la diferencia entre entrada y salida en un programa?**
La entrada (input) son los datos o instrucciones que el usuario proporciona al programa utilizando sentencias de entrada como `Scanner`. La salida (output) es el resultado que el programa muestra en un dispositivo, como el monitor, utilizando métodos como `System.out.print` o `System.out.println`.

**2.- ¿Por qué es importante declarar correctamente el tipo de una variable?**
Es importante porque el tipo determina qué clase de valores puede almacenar la variable (como `int` para enteros, `double` para decimales o `char` para caracteres) y cómo las sentencias de asignación y entrada pueden manipular estos valores durante la ejecución.

**3.- Explica con tus palabras qué es una clase y qué es un método.**
Una clase es una estructura que agrupa una colección de operaciones relacionadas, métodos y miembros de datos, y sirve como unidad básica de un programa Java. Un método es un conjunto específico de instrucciones diseñadas para realizar una tarea particular dentro de esa clase.

**4.- ¿Qué problema resuelve el uso de packages?**
Los *packages* (paquetes) resuelven el problema de la organización del código al proporcionar colecciones de clases relacionadas o bibliotecas ya definidas en Java, como las operaciones matemáticas (`java.lang`) o la clase Scanner para inputs (`java.util`), evitando que el programador tenga que escribir todas las operaciones desde cero.

**5.- Menciona dos reglas de estilo que mejoren la legibilidad del código.**
1. Indentar y dar formato adecuado al código de forma estructurada en lugar de agrupar todo en una sola línea. 
2. Utilizar líneas de comandos explícitas (prompt lines) para decirle de forma clara al usuario qué datos debe introducir y usar espacios en blanco para separar adecuadamente los símbolos y variables.
